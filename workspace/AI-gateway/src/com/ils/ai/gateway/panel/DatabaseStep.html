Clear instance

Drop all tables in a database instance.
Create new database instance

Create tables that support the application. The new tables will be populated with an initial data set. The database must be cleared of any existing tables for this to succeed.
Update an existing instance

Alter previously created tables to bring them in line with the current state of the application. Existing data will be preserved. It should not be necessary to perform this step in addition to the previous.
Populate database instance

Populate tables that support the application or testing of it.
